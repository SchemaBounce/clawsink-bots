## Report Generation

1. Accept the queried dataset and a report configuration specifying sections and grouping fields.
2. Organize records into the configured sections, applying any filters or groupings.
3. Compute summary statistics for each section (counts, totals, averages, min/max).
4. Generate a structured report with title, date range, section headings, data tables, and summaries.
5. Append an executive summary highlighting key findings and notable changes from prior reports.
6. Return the complete report as structured output ready for storage or delivery.
