## Listing Optimization

When optimizing listings:
1. Query str_reviews to extract recurring positive themes and guest-loved features
2. Analyze str_channel_listings for current title, description, and photo order
3. Search semantically for competitor patterns and high-performing listing styles
4. Craft titles: lead with unique differentiator, include location, keep under 50 chars
5. Write descriptions: highlight top 3 amenities, address common guest questions, use platform keywords
6. Recommend seasonal description updates (e.g., ski season vs summer)
7. Update str_properties with optimization suggestions and scores
