## Brand Audit

When auditing brand consistency:
1. Query brand_guidelines from North Star zone for tone, colors, typography, messaging pillars
2. Query recent content_items and brand_assets (created in last 30 days)
3. Evaluate each item against guidelines: tone alignment, visual consistency, terminology usage, messaging clarity
4. Score each item 1-10 on brand compliance and flag items below 6
5. Write brand_findings with specific deviations and corrective suggestions
6. Track brand_scores over time to identify trending drift
7. Escalate systemic issues: message executive-assistant if avg score drops below 7
