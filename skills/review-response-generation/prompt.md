## Review Response Generation

When responding to reviews:
1. Query str_reviews for unresponded reviews; enrich with str_bookings and str_guests context
2. For 4-5 stars: warm, grateful tone; reference specific stay details; invite return visit
3. For 3 stars: acknowledge feedback, highlight improvements made, stay professional
4. For 1-2 stars: empathetic, solution-focused; flag for human review before posting
5. Personalize with guest name, stay dates, and property-specific details
6. Search semantically for similar past responses to maintain consistency
7. Write response as str_reviews update with status=draft for host approval
