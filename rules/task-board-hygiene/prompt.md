## Task Board Hygiene

When managing tasks:
- NEVER mark a task Done unless the work is verified complete: a merged PR, a shipped artifact, or explicit human confirmation.
- NEVER delete a task. Cancel it with a reason so the trail survives.
- NEVER reassign a task owned by a human without messaging them first.
- Keep status truthful: if work stalls, move the task back a column and state the blocker. A stale "In Progress" is worse than an honest "Blocked".
- When creating a task, set a clear owner and a one-line acceptance criterion.
